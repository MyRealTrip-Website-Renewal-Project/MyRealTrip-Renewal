# Button 버튼

실서비스에서 사용하는 다양한 스타일의 버튼 컴포넌트입니다.  
SCSS 모듈 기반, 반응형, 접근성, 다양한 상태(로딩, 비활성, 아이콘 등)를 지원합니다.

## 사용법

```tsx
import { Button } from '@components/ui';

<Button variant="primary" size="md" onClick={...}>확인</Button>
```

---

## 디자인/UX 가이드

- **variant**: primary, secondary, outline, ghost 등 다양한 스타일 지원
- **size**: sm, md, lg
- **icon**: 좌/우 아이콘, 아이콘 전용 버튼 지원
- **loading**: 로딩 스피너 표시
- **fullWidth**: 100% 너비 버튼
- **접근성**: aria-label, 키보드 포커스, role="button" 등 지원

---

## 접근성

- 키보드 네비게이션, 포커스 스타일, aria 속성 지원
