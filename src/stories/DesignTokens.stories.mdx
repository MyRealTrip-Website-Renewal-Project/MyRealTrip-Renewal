# 디자인 토큰/테마 가이드

## 컬러

| 토큰명         | 값         | 용도         |
| -------------- | ---------- | ------------ |
| $primary       | #2196f3    | 주요 색상    |
| $secondary     | #2d3a8c    | 보조 색상    |
| $success       | #4caf50    | 성공        |
| $warning       | #ff9800    | 경고        |
| $error         | #e53935    | 에러        |
| $background    | #ffffff    | 배경        |
| $text-primary  | #232f3e    | 본문 텍스트 |

## 폰트

- Pretendard-Regular, Noto Sans KR 등

## 간격/Radius/그림자
