# MyRealTrip UI 라이브러리

MyRealTrip-Renewal 프로젝트의 통합 UI 컴포넌트 라이브러리입니다.

- **SCSS 모듈 기반 반응형 UI 컴포넌트**
- **Pretendard 폰트, 글로벌 스타일, 접근성, 상태 관리(Zustand)**
- **Storybook 기반 문서/예시/테스트 자동화**
- **실서비스 기준의 디자인 시스템, UX, 반응형, 상태별 스타일, 아이콘, 유효성, 접근성 지원**

---

## 주요 컴포넌트

- **Button**: 다양한 스타일/크기/상태/아이콘/로딩/접근성 지원
- **Input**: 텍스트, 패스워드, 검색, 숫자 등 다양한 입력 필드
- **Card**: 리스트, 배지, 이미지, 섹션 등 다양한 카드 UI
- **Badge, Avatar, List, ProgressBar, Rating, Loader, Toast, Tab, Pagination, Modal, Drawer, Table, Tree, Timeline, Wizard 등**
- **Form, ErrorMessage, Stepper, FileUploader, Filter, DatePicker, Checkbox, Radio, Switch, Select 등**

---

## 사용 가이드

### 설치 및 임포트

```tsx
import { Button, Input, Card } from '@components/ui';
```

### 예시

```tsx
<Button variant="primary" size="md">확인</Button>
<Input label="이메일" placeholder="이메일 입력" />
<Card header="타이틀">카드 내용</Card>
```

---

## 디자인 시스템/UX 원칙

- **일관성**: 컬러, 타이포, 간격, 그림자, radius 등 디자인 토큰 기반
- **반응형**: 모바일/태블릿/데스크탑 대응, 미디어 쿼리/유틸리티 지원
- **상태별 스타일**: hover, active, disabled, loading, error 등
- **접근성**: 키보드 네비게이션, aria, 포커스, 스크린리더 지원
- **유효성/피드백**: 폼 검증, 에러/성공/로딩 메시지, 토스트/알림
- **상태 관리**: Zustand 기반 글로벌/로컬 상태, 커스텀 훅, React Query 등

## Storybook 활용법

- **Docs 탭**: 컴포넌트별 props, 예시, 사용법, 접근성/UX 가이드 확인
- **Controls**: 실시간 props 변경/테스트
- **A11y**: 접근성 자동 검사
- **테스트**: Jest/Vitest 기반 단위 테스트, 스냅샷, 상호작용 테스트

---

## 폴더 구조

- `src/components/ui/` : UI 컴포넌트 모음
- `src/styles/` : 글로벌 스타일, SCSS 변수, 믹스인
- `src/stores/` : Zustand 스토어(상태 관리)
- `src/hooks/` : 커스텀 훅
- `src/types/` : 공통 타입
- `src/utils/` : 유틸리티 함수
